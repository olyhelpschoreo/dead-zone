# DEAD ZONE — Zombie Apocalypse

A browser-based first-person zombie shooter, built from scratch.

**Play it:** https://olyhelpschoreo.github.io/dead-zone/

Around 5,000 lines of hand-written HTML5 Canvas + JavaScript — first-person movement, shooting, and waves of enemies, all in a single file with no game engine or libraries.

*By Nils — a 12-year-old self-taught developer.*
